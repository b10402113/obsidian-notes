# 課程學習路徑總覽

## 📝 課程概述

本單元是整個 Vim 課程的起點。在正式進入 Vim 操作之前，老師先向學員說明這堂課的學習階梯與章節安排，幫助大家建立完整的學習地圖。同時，老師也特別說明課程素材的下載方式，因為所有練習都必須在配套的練習檔案上實際操作，才能真正掌握所學。

---

## 核心觀念與實作解析

### 課程章節的學習階梯

整個課程依難度分為六大區塊，循序漸進：

1. **快速入門（First Steps）** — 最基本的啟動 Vim、進行簡單編輯與儲存的命令。這是讓學員「立刻能用 Vim」的最少必要知識。
2. **用 Vim 的方式思考（Think in Vim）** — 這是本課程的核心精神。一旦學會用 Vim 的思維模式操作，你不需要記憶 dozens of 命令，只需要掌握幾個簡單的 Pattern，就能反覆組合出各種編輯動作。
3. **內建幫助系統（Built-in Help System）** — 學會查文件，讓你在課程結束後依然能自給自足，不依賴外界資源。
4. **文字替換與全域變換（Transforming & Substituting）** — 從簡單替換起步，逐步做到跨多個檔案一次全域取代。
5. **強大進階功能（Advanced Features）** — 包括 Text Objects、Macros、各類設定的自訂，以及同時編輯與檢視多個檔案的技巧。
6. **圖形介面 Vim（GUI Vim）** — 介紹 Vim 的圖形化版本，以及滑鼠、選單、系統剪貼簿整合等純文字模式沒有的功能。

> **為什麼要先建立學習地圖？** 因為知道自己在整條路徑中的哪個位置，學習才會有方向感。Vim 的學習曲線雖然看起來陡峭，但只要按部就班，每個階段都不會太吃力。

### 課程素材下載與環境準備

老師在影片中特別強調，**跟著做（follow along）是學習 Vim 最重要的事**，因此必須先完成環境準備：

1. 在影片右上角點擊「Resources available」，下載名為 `Vim Class Zip` 的壓縮檔
2. **macOS** — 雙擊 zip 檔，自動解壓縮至 `Vim class` 資料夾
3. **Windows** — 對 zip 檔按右鍵，點擊「Extract All」完成解壓縮
4. **Unix / Linux** — 執行 `unzip -d vim-class vim-class.zip`

解壓縮後，所有上課示範用檔案與練習題說明都會在 `Vim class` 資料夾中。**練習題說明檔案名稱以「Exercise」開頭並附有編號**（例如 `Exercise 01`），每個練習檔內會清楚說明需要搭配哪些其他檔案才能完成。

---

## 💡 重點摘要

- **Vim 的學習核心不在於背命令，而在於學會「用 Vim 的思維」組合操作。**
- **「Think in Vim」是本課程最重要的核心理念，一旦掌握，後續所有功能都會自然連貫。**
- **所有練習都必須下載課程壓縮檔，在配套檔案上實際操作，而非只看影片。**
- **練習題說明檔以「Exercise + 編號」命名，檔案內會標明需要搭配的資料。**
- **課程從「最小必要知識」起步，逐步過渡到進階功能，學習坡度設計平緩。**
