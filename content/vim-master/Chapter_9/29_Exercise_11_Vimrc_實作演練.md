# Exercise 11：Vimrc 實作演練

## 📝 課程概述

本單元是一堂完整的 Vimrc 實作練習。老師帶領學員從無到有，一步步完成個人專屬的 vimrc 檔案：先閱讀課程提供的範本檔案、了解每個設定的用途，接著根據個人需求啟用或註解掉不需要的選項，最後將調整後的內容複製到自己的 vimrc 中。

---

## 核心觀念與實作解析

### 練習前置準備

1. 開啟終端機，找到課程下載資料夾中的 **RC 範本檔**（例如 `class/vimrc_sample`）
2. 用 Vim 開啟該檔案

```bash
cd ~/Downloads/class
vim vimrc_sample
```

> 這裡假設課程下載檔案已解壓縮到 Downloads 資料夾。實際路徑視個人下載位置而定。

---

### 第一階段：閱讀、測試、決定取捨

千萬不要在沒看過說明的情況下就把所有設定全部打開。建議流程是：

**步驟 1：先通讀一遍，邊讀邊做決策**

每一個設定前通常都有註解說明該選項的用途。例如：

```vim
" ruler: Show the line and column number of the cursor position
set ruler
```

如果這個說明讓你聯想到自己可能會用到，**先不要急著啟用**，而是去查 Vim 內建的說明文件。

**步驟 2：用 `:help` 查詢該選項的詳細說明**

```vim
:help 'ruler'
```

> 使用**單引號**包住選項名稱，會直接跳到該選項的專屬說明文件，而不是跑到其他相關主題。

**步驟 3：實際啟用看看效果**

```vim
:set ruler
```

 Vim 會立即在右下角顯示游標位置。看完效果後，決定要保留或關閉。

**步驟 4：如果決定不用，直接註解掉**

```vim
" set ruler
```

用 `shift+I`（大寫 I）在行首插入 `" ` 將該行註解，這樣 Vim 執行 vimrc 時就不會執行該設定。

**步驟 5：每次修改後立即存檔**

```vim
:w
```

養成「每做一個變更就存一次檔」的習慣，避免修改流失。

---

### 查閱說明文件的延伸技巧

課程範本裡提到一個很實用的查詢方式：

```vim
:options
```

這個指令會開一個視窗，列出**所有 Vim 選項及其簡短說明**，你可以直接在裡面瀏覽、啟用或停用選項，適合想要系統性地認識所有選項的學習階段。

---

### 第二階段：將範本複製到自己的 vimrc

假設你對範本已經做完了個人化調整，現在要把這份內容搬到自己的 vimrc 檔案中。整個過程完全在 Vim 內完成，**適用於所有作業系統**。

**步驟 1：在範本檔案中，將全部內容 yank 到一個暫存 register**

```vim
gg          " 跳到檔案最開頭
"vyG        " 使用 v register，yank 從目前位置到檔案結尾的所有內容
```

> 老師特別選 `v` register（想像成 "vimRC" 的縮寫），方便記憶。yank 完成後，Vim 會顯示一共 yank 了多少行（例如 `86 lines yanked`）。

**步驟 2：開啟自己的 vimrc 檔案**

| 作業系統 | 指令 |
|---|---|
| Linux / macOS | `:edit ~/.vimrc` |
| Windows | `:edit ~/_vimrc` |

```vim
:edit ~/.vimrc
```

**步驟 3：貼上剛才 yank 的內容**

```vim
"VP    " 大寫 P：將 register 內容貼在游標**上方**
```

> 如果用小寫 `p`，內容會貼在游標下方，檔案頂端會多出一行空白，兩者皆可，只是美觀問題。

**步驟 4：存檔離開**

```vim
:wq!
```

---

### 為什麼要親自做這份練習？

這堂練習的核心價值在於：

1. **強迫你一個一個去認識那些設定**：不像老師直接告訴你「用這四個就好」，你需要自己查文件、自己判斷，這個過程會大幅加深對 Vim 選項系統的理解。
2. **建立「個人化」的起點**：每個人的使用情境不同，有人需要行號，有人需要備份，有人寫 C 語言，有人寫 Python——這份 vimrc 就是你日後持續調整的起點，隨著經驗累積再慢慢增減。
3. **學會使用 `:source`**：如果日後想要微調自己的 vimrc，修改後不必每次都重啟 Vim，用 `:source ~/.vimrc` 就可以立即在當前 Vim 中測試效果。

---

## 💡 重點摘要

- **使用 `:help 'optionname'`（帶單引號）直接查詢該選項的專屬說明，避免被其他相似主題干擾。**
- **不想要的設定不要刪掉，用雙引號 `"` 註解掉就好，之後想回顧時還能看得到原始說明。**
- **用 Vim 的 register 機制（`:edit ~/.vimrc` + `"vP`）將範本內容轉移到自己的 vimrc，適用於所有作業系統。**
- **每次修改後立即 `:w` 存檔，養成良好習慣。**
- **`:options` 指令可以打開一個互動式選項視窗，是認識所有 Vim 設定的好工具。**
